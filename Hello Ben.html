Hello Ben,

As you might be aware I’m not familiar with html and didn’t have enough time to spend on learning bootstrap.

Because of this, my CA2 looks very simple and does not have many features.

It has very basic bootstrap functionality, like: navbar, grid, images, alerts, buttons, glyphicons, list, collapse, scrollspy. 

As you requested last time, I modified a bit the Education and Experience sections. I tried to use carousel, modal, popover and tooltips for Education page but have no success. The features were not functioning properly. The only way to make it work at least somehow was to use a screen centred pop-up window with predefined size, which is opening for now with a picture in it describing my education results. Within the Education section I used picture thumbnails which are in fact links to the institutions’ web page. Text under pictures is used as link to the pop-up window. If I would have more time, I would use modal feature for this purpose and would extend the description and achievements within the Education section.

I would also add some interactive features, add skills section and make the messaging feature fully functional.
